Qual é a diferença entre HTML, CSS e JavaScript?

Diferença entre HTML e CSS:

HTML: linguagem de marcação utilizada para estruturar os elementos da página, como parágrafos, links, títulos, tabelas, imagens e até vídeos. CSS: linguagem de estilos utilizada para definir cores, fontes, tamanhos, posicionamento e qualquer outro valor estético para os elementos da página.

Diferença entre Linguagens de Marcação e JavaScript:

As Linguagens de marcação são usadas para apresentar informações, enquanto o JavaScript é uma linguagem de programação que permite criar interatividade em páginas web.
